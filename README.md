# skills_set

Reusable Codex skills maintained in this repository.

## Skills

- `orchestrator-workers`: Coordinate a GPT-5.5 planning agent with worker53 GPT-5.3 Codex subagents for coding, debugging, diagnosis, verification, and other heavy execution work.
- `professional-book-reading-tutor`: Turn professional books, textbooks, screenshots, scanned pages, formulas, and dense technical sections into structured study notes, derivations, and review cards.

## Layout

Each skill is stored in its own directory and includes a `SKILL.md` file. Additional references or assets live inside that skill directory.
